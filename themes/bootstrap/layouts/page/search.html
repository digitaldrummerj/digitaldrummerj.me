Advanced Search Options:

  • Wildcards: *
  • Keyword Prescence: Default is OR multiple words.
    • Include Term: add + to start of term (e.g. +aspnet
    • Exclude Term: add - to start of term (e.g. -dotcover
    • You can combine them together. +foo bar -baz must contain foo, might contain bar and must not contain baz
  • Dashes in Term: if you use a dash as part of the term, it must be escaped with a \ e.g. aspnet\-core
  • Filter by Category or other Field: you can tell the search to include a term in a field such as a category or title.
    • categories:aspnet\-core to search the aspnet-core category
    • Available fields are: title, subtitle, categories,description
